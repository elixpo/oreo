# apps_icon — Home screen "Apps" launcher button icon

**Asset name:** `apps_icon`  
**Output size:** 200×200 → optimized to 32×32  
**Used in:** `lix_os/home.py` — the single dock button that opens the app menu

## Prompt

pixel art 8-bit icon: a 3×3 grid of nine tiny colourful square app tiles on a very dark near-black background, each small tile a different bright neon colour (hot pink, teal, amber gold, soft green, purple, orange, cyan, magenta, white), thick dark pixel outlines between tiles, chunky retro 8-bit pixels, clean square composition, no text, no characters, white background, thick dark pixel border around whole grid

## Theme notes

- Dark background inside the icon: rgb(14, 12, 20)
- Tile accent colours should include the mascot palette: pink rgb(255,93,104), teal rgb(0,180,165), gold rgb(255,200,60)
- White background for the outer PNG (optimizer will composite onto dark badge bg)
